## Tech Stack

- Next.js 15
- React 18
- Tailwind CSS (using `@tailwindcss/postcss` plugin)
- Framer Motion
- Three.js / React Three Fiber

## Local Development

```bash
npm install
npm run dev
```
App will start on `http://localhost:3000` (or the next free port).

## Production Build

```bash
npm run build
npm start
```

## Scripts

- `dev`: Run local development server
- `build`: Create production build
- `start`: Run production server
- `lint`: Lint (currently skipped in build via config)

## GitHub Setup

Initialize and push to a new GitHub repository:

```bash
git init
git add .
git commit -m "Initial portfolio"
git branch -M main
git remote add origin https://github.com/<your-username>/<repo-name>.git
git push -u origin main
```

Replace `<your-username>` and `<repo-name>` accordingly. If the remote exists already, skip `git init` and `git branch -M main`.
